# `kattis` competitive programming problems

This repo holds solutions to Kattis problems I've solved, as well as some notes
on both the final solution I reached for the harder ones, as well as prior
attempts to reach the solution.

The problems are organized by chapters, outlining the problem set that
corresponds with those recommended at [[1]] for each of the chapters on the
books _Competitive Programming 4: Book 1_ and
_Competitive Programming 4: Book 2_.

[1]: https://cpbook.net/methodstosolve?oj=kattis&quality=starred&difficulty=all
